# reactapp with gulp & webpack

##开发环境
###安装依赖(在开始之前请安装node.js、gulp、bower)
`cd path/to/reactapp`

`npm install`

`bower install`

##启动服务
`npm start`

##开启实时编译
`cd path/to/reactapp` (open a new terminal window or tab)

`gulp compile`

[http://localhost:1080/](http://localhost:1080/)

##发布
`gulp build`
