# reactapp + antd with gulp & webpack

##开发环境
###安装依赖(在开始之前请安装node.js)
`cd path/to/reactapp-antd`

`npm install`
`

##开启实时编译
`cd path/to/reactapp`

`gulp watch`


##启动服务
`npm start`  (open a new terminal window or tab)

[http://localhost:2048/](http://localhost:2048/)

##打包构建
`gulp build`
